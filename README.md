🛍️ E-Commerce Recommendation System

📌 Overview
This project is a **content-based recommendation system** that suggests similar products using Machine Learning techniques. It analyzes product features like category and description to find and recommend related items.

---

🚀 Features
* 🔍 Product recommendation based on similarity
* 🧠 TF-IDF vectorization for text processing
* 📊 Cosine similarity for matching products
* 🎯 Category-based filtering
* 💻 Interactive web app using Streamlit
* 📈 Displays similarity score (%)

---

🧠 Tech Stack

* Python
* Pandas
* Scikit-learn
* Streamlit

---

📂 Project Structure

Ecommerce-Recommendation-System/
│── app.py

│── README.md

---

▶️ How to Run

1. Install required libraries:

```bash
pip install pandas scikit-learn streamlit
```

2. Run the application:

```bash
streamlit run app.py
```

3. Open in browser:

```
http://localhost:8501
```

## 📷 Output

* Select a category
* Choose a product
* Get top 5 similar product recommendations with similarity score
